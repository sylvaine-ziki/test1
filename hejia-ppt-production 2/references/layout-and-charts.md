# 版式与图表规则

## 版式选择

| 信息意图 | 推荐版式 |
|---|---|
| 解释一个核心判断 | 左结论 + 右证据，或上结论 + 下证据 |
| 展示 3–5 个并列要点 | 等宽卡片、多栏矩阵 |
| 展示过程或机制 | 横向流程、闭环、分层架构 |
| 展示时间演进 | 时间轴、柱线组合图 |
| 展示差异与比较 | 柱形图、条形图、对比矩阵 |
| 展示构成 | 饼图、环形图、100% 堆积柱形图 |
| 展示建议 | 编号列表、优先级矩阵、路线图 |

版式可以创新，但必须保留固定 Storyline 和数据来源区域。

## 柱形图 / 条形图

- 排序应服务于结论：比较排名时通常降序；时间序列通常升序。
- 主系列使用主红；对照系列使用蓝；第三系列使用橙。
- 非重点柱可使用灰色或主色浅阶，重点柱使用主色。
- 删除默认网格线和默认标题。
- 坐标轴从 0 开始，除非截断有明确理由并作注释。
- 显示关键数据标签；标签过密时仅保留最大值、最小值、最新值或重点值。
- 图表上方使用一个居中文本框放置标题和单位；标题第一行，单位第二行。

## 饼图 / 环形图

- 只用于表达构成，类别通常不超过 6 个。
- 按占比从大到小排列；“其他”放最后。
- 最大或最关键扇区使用主红，第二类使用蓝，第三类使用橙，其余使用灰色阶。
- 显示类别名和百分比；避免只依赖图例。
- 占比差异难以观察时，改用排序条形图。
- 环形图中央可放总量或核心结论。

## 折线图 / 柱线组合

- 折线用于趋势，柱形用于规模；不要无理由使用双轴。
- 使用右轴时必须在系列名称中标注“（右轴）”。
- 标记转折点、异常值与最新值，重点标记不超过 3 处。

## 图表通用格式

- 页面相对空旷或本页只有一个图表时：标题第一行 14 pt、单位第二行 12 pt，均使用思源黑体 CN Regular 字体文件并设置加粗属性、`#404040`。
- 图表标题和单位统一使用 120% 行距。
- 图例和标签：10.5–12 pt，思源黑体 CN Regular；重点标签仅设置加粗属性，禁止使用 Bold 字体文件。
- 数字默认保留一位小数；仅将一位小数显示发生碰撞的数据组升级为两位小数，其他数据仍保留一位。
- 单位统一；不使用无意义千分位。
- 图表必须可编辑，禁止用 Excel 截图或图片冒充图表。
- 源数据应保留在图表数据、备注或画布外核查区。
