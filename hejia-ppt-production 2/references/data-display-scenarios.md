# 数据展示与场景决策规则

## 1. 全局规则

以下规则适用于所有图表、表格和大段文字块，不因场景改变：

- 每个内容块必须有标题。
- 标题和单位必须放在同一个文本框内，并位于内容块上方居中。
- 标题和单位统一使用 120% 行距。
- 页面相对空旷或本页只有一个图表时：
  - 第一行标题：14 pt，思源黑体 CN Regular 字体文件并设置加粗属性。
  - 第二行单位：12 pt，思源黑体 CN Regular 字体文件并设置加粗属性。
- 标题命名优先使用：`年份 + 地区 + 主体`。
- 示例：`2020-2025年全球磷矿石供需对比`。
- 数据来源始终为 10 pt、1.0 倍行距，固定以“，和君咨询分析”结尾，不添加小三角。
- 图表、表格和大段文字块中的所有内容统一使用思源黑体 CN Regular；需要强调时仅设置加粗属性。

## 2. 数字小数位

- 原则上数字显示一位小数。
- 当多个数据保留一位小数后显示相同、无法辨别差距时，仅将发生显示碰撞的那组数据升级为两位小数。
- 未发生碰撞的数据继续保留一位小数，不将整张图统一改为两位。
- 示例：`0.12、0.13、0.8` 显示为 `0.12、0.13、0.8`。
- 示例：若一组数据中只有两个数在一位小数下都显示为 `0.1`，仅这两个数显示两位小数，其他数仍显示一位。
- 百分比、倍数、金额和规模数据同样遵守该规则，但必须统一单位。

使用 `scripts/data_display_rules.mjs` 中的 `formatAdaptiveNumbers()` 实现。

## 3. 多指标场景

多指标场景不是固定版式，应根据本页结论、空间和数据关系灵活选择：

- 单一年份的规模型指标，例如营收、利润、销量，可优先使用柱形图或条形图。
- 多个企业对比时，可按一个关键规模指标降序排列；排序指标必须服务于本页结论。
- 增速、CAGR、利润率和变动值可与规模指标拆分展示，避免将不同量纲强行挤在同一张图。
- 当图形化会造成信息损失时，使用带两级表头的表格。
- 指标组过多或对象过多时拆页，不通过缩小字号解决。

## 4. 企业业绩场景示例

若数据包括营收、营收 CAGR、营收同比增速、毛利率、毛利率变动、毛利率同比变动，可选择：

- 将单一年份营收按规模降序绘制柱形图。
- 将毛利率单独绘制排序柱形图或点图。
- 将 CAGR 与同比增速作为独立图表、标签或表格列。
- 使用两级表头表格：第一层为“营收”“毛利率”等指标组；第二层为具体年份、CAGR 区间、同比增速区间等字段。

上述内容是场景策略，不是强制每次都使用相同版式。

### 企业业绩表格固定指标顺序

企业业绩表格的指标组及列顺序是硬性规则，禁止根据版面、排序或视觉偏好调整：

1. 营业收入：`2024年数据 → CAGR → 2025H1同比增速`。
2. 销售毛利率：`2024年数据 → 变动 → 2025H1变动`。
3. 市值相关数据：`总市值 → PE → PB`。

- 表格必须使用两级表头：第一层为“营业收入 / 销售毛利率 / 市值相关数据”；第二层严格按照上述字段顺序。
- 企业业绩表格默认必须在一页内完整展示；仅当用户明确要求拆页时才允许拆页。
- `25H1同比`列只展示同比增速，不展示 2025H1 营收数据。
- `25H1变动`列只展示毛利率变动，不展示 2025H1 毛利率数据。
- 毛利率具体年份数据使用 `%`；毛利率的所有“变动”字段统一使用 `pct`，包括历史变动和 H1 变动。
- 市值相关数据默认仅展示总市值、PE、PB，不单独展示其同比增速列；表头禁止添加 PE/PB 后的括号及括号内容。
- 禁止将市值、PE、PB放在营业收入和销售毛利率之前。
- 企业或业务环节的排序可以服务于结论，但不得改变指标列顺序。
- 在企业业绩表格中，`营业收入-2024`和`销售毛利率-2024`允许优先改为单元格内横向数据条，其他指标继续使用数字文字。
- 数据条长度表达数值大小，必须保留数值标签、同列统一起点和统一刻度；长数据条的标签可放在条内，短数据条的标签放在条右侧。
- 禁止使用无法清晰比较长度的细小竖柱。
- 企业业绩页需要展示“企业经营概况”模块，概括业务环节、代表企业及其主营业务构成；概况模块不得改变业绩表格固定指标顺序。
- 企业名称、主营构成和右侧业绩指标必须处于同一水平行，并使用贯穿整行的分隔线或统一行底色形成强对齐网格。
- 企业经营概况不得作为与右侧表格行无关的独立摘要卡片。
- 企业经营概况优先采用“最左侧纵向业务环节分组标签/箭头 + 企业名称色块 + 主营构成 + 右侧业绩指标”的布局；业务环节标签可跨越多行，不需要在每行重复。
- 企业业绩表格必须展示用户提供的全部企业，禁止擅自筛选、删除或仅展示代表企业。
- 企业业绩表格字体可以为适配单页进行调整，但任何正文、表格、标签和数据不得低于 10.5 pt。
- 一页内容较多时，优先压缩留白、行高和非必要装饰，同时保持严格对齐，不得删除企业。
- 表头采用指标名称与口径/年份两层结构；允许将 `2024年数据`简写为`2024`，将`2025H1同比增速`简写为`25H1同比`。
